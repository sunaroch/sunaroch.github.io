Miscellaneous Music

WHAAAAAA: This is a piece that I made with ableton.

Percussive: This is a piece that I created during my time studying abroad at Goldsmiths, with samples from instruments made by Hugh Davies. I primarily used Logic Pro X.

Wishwash: This is another piece that I created during my time studying abroad, I recorded my own voice, and then heavily effected the recording using Logic Pro X.

Detroit Techno Meets Live Coding: After learning about Detroit Techno in a class called "The History and Practice of Electronic Music," I was inspired to create my own rendition, but with a twist. For the drums, I utilized Sonic Pi, a programming language designed for live coding. Using these drums, I built and mixed the rest of the track using Logic Pro X.

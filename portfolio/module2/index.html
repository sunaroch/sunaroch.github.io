Sun's Module 2:

Sun Synth!

My goal for this project was to create an instrument using MaxMSP. I knew that I wanted a potentiometer to affect frequency, and a button to turn the instrument on and off.

The primary hardware function (and the most difficult part, in my opinion) is reading the values from the button and the potentiometer, and sending an OSC message to MaxMSP with a python backend. Once I had these values in MaxMSP, I had full creative freedom.

I wanted to create a bright and interesting sound, so I added two sine waves together with an added delay. One was a mapped value from 0 to 1024, and the other was a mapped value from 256 to 0. After adding the delay, I was happy with the sound I had created.

In order to turn the button into an on/off switch, I selected only the 1 values from the button, and loaded that into a bang and then into a toggle object.
